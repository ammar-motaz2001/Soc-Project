# نظام الإجراءات التلقائية (Automated Actions)

## 📋 نظرة عامة

تم تحديث نظام IRAS ليتضمن إجراءات تلقائية ذكية تُنفذ تلقائياً عند ظهور تنبيهات أمنية بمستوى **Critical** أو **High**.

## ⚡ كيف يعمل النظام

### 1. الكشف التلقائي
- عند ظهور أي Alert بمستوى **Critical** أو **High**
- يتم تشغيل نظام الإجراءات التلقائية فوراً
- يظهر إشعار Toast في أعلى يمين الشاشة

### 2. الإشعارات الفورية
```
🚨 Critical Alert Detected!
Alert 8820: Suspicious PowerShell Execution Detected
Automated actions initiated.
[View Actions]
```

### 3. الإجراءات المنفذة تلقائياً

#### لجميع التنبيهات عالية الأولوية:
1. **إرسال إشعارات** (Notification)
   - Email للفريق الأمني
   - رسالة Slack
   - SMS للمحلل المناوب

2. **تحليل التهديدات** (Threat Intelligence)
   - استخراج مؤشرات الاختراق (IOCs)
   - البحث في VirusTotal
   - التحقق من قواعد بيانات التهديدات

3. **ارتباط السجلات** (SIEM Correlation)
   - جمع 45,000+ سجل ذو صلة
   - بناء جدول زمني للهجوم
   - إنشاء تقرير الحادثة

#### إجراءات خاصة حسب نوع التهديد:

**🖥️ Endpoint Threats:**
- عزل الجهاز من الشبكة (Critical فقط)
- إيقاف العمليات الخبيثة
- جمع ذاكرة Forensic

**🌐 Network/Firewall Threats:**
- حظر عنوان IP المشبوه
- تحديث قواعد الجدار الناري
- نشر الحظر على 3 شرائح شبكة

**📧 Phishing:**
- عزل البريد الإلكتروني
- البحث في جميع صناديق البريد
- إزالة الرسائل الخبيثة تلقائياً

## 📊 صفحة Automated Actions

### الإحصائيات المتقدمة
- إجمالي الإجراءات المنفذة
- معدل النجاح
- الإجراءات المعلقة والفاشلة
- متوسط وقت التنفيذ
- إجمالي استدعاءات API

### الرسوم البيانية
- **Pie Chart**: توزيع الإجراءات حسب الفئة
  - Containment (الاحتواء)
  - Investigation (التحقيق)
  - Notification (الإشعارات)
  - Remediation (المعالجة)

- **Bar Chart**: حالات التنفيذ
  - Success (نجح)
  - Pending (معلق)
  - Failed (فشل)
  - Partial Success (نجح جزئياً)

### الجدول التفصيلي
كل صف في الجدول قابل للتوسع ويعرض:
- معرف التنبيه
- نوع الإجراء
- الفئة والحالة
- وقت ومدة التنفيذ
- عدد استدعاءات API
- Playbook المستخدم

### التفاصيل الموسعة
عند النقر على أي إجراء، تظهر:
- **العمود الأيسر:**
  - قاعدة التنبيه
  - الوصف التفصيلي
  - الأصول المتأثرة
  - من قام بالتشغيل
  - حجم البيانات المعالجة
  - رسائل الأخطاء (إن وجدت)

- **العمود الأيمن:**
  - جدول زمني تفصيلي للتنفيذ
  - حالة كل خطوة (✓ مكتمل / ⏱ معلق / ✗ فشل)
  - مدة كل خطوة
  - تفاصيل كل خطوة

### الفلاتر المتقدمة
1. **البحث النصي**: بحث في نوع الإجراء، قاعدة التنبيه، أو معرف التنبيه
2. **فلتر الحالة**: عرض إجراءات محددة (Success/Failed/Pending)
3. **فلتر الفئة**: تصفية حسب نوع الإجراء

## 🔄 تدفق العمل

```
Alert Detected (Critical/High)
    ↓
Automated Actions Triggered
    ↓
1. Send Notification (0.3s) ✓
2. Threat Intelligence (3.4s) ✓
3. Containment Action (1.2s) ✓
4. SIEM Correlation (5.2s) ✓
    ↓
Toast Notification Shown
    ↓
Actions Logged in Database
    ↓
Visible in Automated Actions Page
```

## 🎯 الفوائد

1. **استجابة فورية**: تنفيذ خلال ثوانٍ من الكشف
2. **تغطية شاملة**: إجراءات متعددة لكل تهديد
3. **شفافية كاملة**: تتبع تفصيلي لكل خطوة
4. **احترافية عالية**: واجهة احترافية مع تفاصيل شاملة

## 📱 الإشعارات

- تظهر في أعلى يمين الشاشة
- تبقى لمدة 8 ثوانٍ
- يمكن النقر على "View Actions" للانتقال مباشرة
- تتضمن رقم ونوع التنبيه

## 💡 ملاحظات

- الإجراءات التلقائية تُنفذ فقط للـ **Critical** و **High**
- التنبيهات المتوسطة والمنخفضة تتطلب مراجعة يدوية
- جميع الإجراءات مسجلة بشكل دائم
- يمكن تصدير السجلات للمراجعة والامتثال
